# ♟️ Chess Browser

A web based chess database browser built with C#, Blazor, and MySQL. This project lets users upload PGN files, store chess games in a database, and search through matches using filters such as player names, opening moves, winners, and date ranges.

---

## 📦 Stack

- C#
- ASP.NET Core Blazor
- Razor Components
- MySQL
- Bootstrap

---

## ✨ Features

- Upload and parse PGN chess files
- Store players, events, and games in a MySQL database
- Search games by white player, black player, opening, winner, and date range
- Display event info, player names, ratings, results, and moves
- Progress tracking during PGN uploads
- Simple web interface for browsing chess data
